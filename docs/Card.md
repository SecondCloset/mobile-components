# Card

A standard card component.

| prop     | isRequired? | dataType        | possibleValues           |
| -------- | ----------- | --------------- | ------------------------ |
| type     | no          | string          | "box"                    |
| children | yes         | React.component | <Text>Hello World</Text> |
